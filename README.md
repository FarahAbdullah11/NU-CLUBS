# NU Clubs Hub

Single-page React + TypeScript hero section for the Nile University Clubs & Student Union portal.

## Getting started

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:5173/`.

## Build for production

```bash
npm run build
npm run preview
```

`npm run preview` serves the generated assets so you can do a final visual check before deployment.
